# Справки, заявления

Сервис по оформлению справок, заявлений и запросов Московского Политеха.

Запуск проекта на локалхосте:

```
cd mpu-statements-service
pip install -r requirements.txt
python manage.py runserver
```

или

```
cd mpu-statements-service
docker-compose build
docker-compose up -d
```
